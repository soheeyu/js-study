컴포넌트에 CSS를 적용하는 방법을 학습한다. 컴포넌트에 인라인 스타일링을 사용한다. 작업 순서 1. Letter 컴포턴트 만들기. 아래의 html 코드를 컴포넌트로 만들기 <div class="letter" style="background-color:#58B3FF;">A</div> 2. Letter 컴포넌트에서는 background-color 값을 부모에서 bgcolor 속성값을 넘기면 받아서 사용하도록 한다. 예시) <Letter bgcolor={"#58B3FF"}><span>A</span></Letter> 3. 자식 태그를 넘겨 받는 방법은 this.props.children 을 사용한다. slot을 받는 방법 예시) <Letter bgcolor={"#58B3FF"}> <span>A</span> <== this.props.children 을 사용하여 받을 수 있다. </Letter> 4. { padding: 10px; margin: 10px; color: #333333; display: inline-block; font-family: monospace; font-size : 32px; text-align: center; } + { background-color: black; } <div style={...this.state.styl1, ...{backgroundColor: this.props.bgcolor} }> </div> 4. ReactDOM 을 사용하여 화면에 붙인다. 예시) ReactDOM.render( <div> <Letter bgcolor={"#58B3FF"}> A </Letter> <Letter bgcolor={"#FF605F"}><span>E</span></Letter> <Letter bgcolor={"#FFD52E"}><span>I</span></Letter> <Letter bgcolor={"#49DD8E"}><span>O</span></Letter> <Letter bgcolor={"#AE99FF"}><span>U</span></Letter> </div> , document.getElementById("app") );
